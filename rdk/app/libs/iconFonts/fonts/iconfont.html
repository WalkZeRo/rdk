使用说明
第一步：使用font-face声明字体
注: font-face引用的四个字体文件的路径以及文件名(eot/woff/ttf/svg)需要根据实际资源位置修改
@font-face {
    font-family: "iconfont";
    src: url("iconfont.eot"); /* IE9*/
    src: url("iconfont.eot?#iefix") format("embedded-opentype"), /* IE6-IE8 */
    url("iconfont.woff") format("woff"), /* chrome、firefox */
    url("iconfont.ttf") format("truetype"), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url("iconfont.svg#iconfont") format("svg"); /* iOS 4.1- */
}
第二步：定义使用iconfont的样式 
.iconfont{
    font-family:"iconfont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;
}
第三步：挑选相应图标并获取字体编码，应用于页面 
 <i class="iconfont">&#x1005;</i>
附录1: 动态修改图标引用
如果您需要在页面上动态引用或者改变图标的引用, 请使用 class 的方式引用图标, 使用纯 lable 的形式修改后会变为 '口' 字样的乱码
需要在页面额外引用该目录下同步输出的 CSS 文件, 然后根据对应的图标代码选用相应前缀为 'iconfont' 的 class
例如: 图标代码为 '&#x1005;' 的图标, 您需要将i标签内容留空, 然后为其应用 'iconfont'、'iconfont-1005' 这两个 class
其中 'iconfont-1005' 这个 class 对应原图标字码中的 '&#x1005;' 中的 '1005' 部分
 <i class="iconfont iconfont-1005"></i>
附录2: 直接引用环境中的RDK内的图标字体文件
如果您想要在 VMAX 环境中引用该 iconfonts, 请确认您当前所在的环境中已经部署了 RDK. 目前, 图标字体与 RDK 打包发布
确认是否可以引用 iconfont 的最快方式, 您可以尝试访问环境下的 'rdk_server/app/libs/iconFonts/css/iconfont.css' 目录
顺带一提, 如果您需要动态修改图标引用的话, 请引用 'rdk_server/app/libs/iconFonts/fonts/iconfont.css' 文件
如果您需要新增图标, 先确认这里是否包含您所需要的图标, 如果没有, 可以访问 RDK 的 Gitlab 项目来提一个 ISSUE 告诉我们.
