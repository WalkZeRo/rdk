简介

rdk_map 主要用于Echart版本的地图展现

这是一个简单的 rdk_map 例子：

属性

id

支持类型：字符串

id 用于唯一标识一个图形，主要用于事件交互。

map_define

map_define 用于指定生成图形的模板文件Url

width

width 用于指定图形的宽度。

height

height 用于指定图形的高度。

data

data 用于生成图形显示的其他数据。

map

map 用于指定地图资源文件地址,目前仅支持json文件格式，该类文件可从echart官网下载http://echarts.baidu.com/download-map.html。

事件

接收的事件

UPDATE_GRAPH

接收的事件，用于外部数据变化后，发事件通知地图更新。

mapSelect/mapUnSelect

接收的事件，用于外部选择或者取消地图的某块区域

发出的事件

  1. click
  2. dblclick
  3. mousedown
  4. mouseup
  5. mouseover
  6. mouseout
  7. globalout
  8. legendselectchanged
  9. legendselected
  10. legendunselected
  11. datazoom
  12. datarangeselected
  13. timelinechanged
  14. timelineplaychanged
  15. restore
  16. dataviewchanged
  17. magictypechanged
  18. pieselectchanged
  19. pieselected
  20. pieunselected
  21. mapselectchanged
  22. mapselected
  23. mapunselected

具体使用场景

地图点击

切换地域

外部选择某块区域

不同地域渲染成不同颜色块并加上气泡

地域砖取
