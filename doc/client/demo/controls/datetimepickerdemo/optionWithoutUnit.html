不带粒度选择的时间控件

1、为时间控件设置初始值

In the HTML:

In the Controller:

scope.defaultValue = { value: "2015-01-01 07:00", };

2、为时间控件设置可点击日期范围

In the HTML:

In the Controller:

scope.setLimitValue = { value: "2015-01-07 07:00", startDate:"2015-01-01 07:00", endDate:"2015-01-09 07:00" };

3、可以直接设置时间控件的粒度

In the HTML:

In the Controller:

scope.unitSet = { value: "2015-01-07 07:00", startDate:"2015-01-01 07:00", endDate:"2015-03-09 07:00", gr:"week" //hour day week month range };

4、时间控件粒度为range时的输入值

In the HTML:

In the Controller:

scope.rangeSet = { value: ["2015-01-07 07:00", "2015-01-17 07:00"], startDate:"2015-01-01 07:00", endDate:"2015-03-09 07:00", gr:"week" //hour day week month range };
